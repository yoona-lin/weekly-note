- **076.精读《谈谈 Web Workers》**
  > Web Workers 用来执行异步脚本，只要掌握了它与主线程通信的方式，就可以在指定时机运行异步脚本，并在运行完时将结果传递给主线程。
  > Web Workers 的两大特征：高效、并行。
